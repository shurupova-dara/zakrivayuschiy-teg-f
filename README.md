Учебный проект по вёрстке адаптивного сайта с карточками и интерактивными элементами (лайки, диалоговое окно сохранения).

 О проекте

Проект выполнен в рамках обучения верстке.  
Основная цель — закрепить навыки:

- семантической HTML-разметки
- работы с CSS (flex, позиционирование, псевдоклассы)
- SVG-иконок и спрайтов
- анимаций
- модального окна `<dialog>`
- структуры проекта и чистоты кода

Функционал

- карточки с изображениями и текстом
- кнопка лайка с SVG-анимацией
- кнопка «Сохранить на память» с открытием диалога
- модальное окно с сообщением
- использование SVG-спрайта (иконки)

 Особенности реализации

- стили разделены по файлам:
  - `globals.css`
  - `variables.css`
  - `style.css`
  - `animations.css`
- SVG используется через `<use>`
- применены CSS-фильтры к изображениям (минимум 7 вариантов)
- уникальные классы для каждой карточки изображения
- адаптация под макет
